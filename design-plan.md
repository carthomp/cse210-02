# Design Plan

## Director
---
This class will do the following things using appropriate methods:
```
start method -  
+-- Starts the game with 300 points

turn method -   
+-- Picks a card
+-- Asks the user for a guess
+-- Picks the next card
+-- Compares the two cards
+-- Updates the score

end method -
+-- Asks to play again if score > 0
```

pick method -
